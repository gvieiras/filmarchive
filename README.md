# filmarchive
## ENG
Personal project developed to practice studied concepts. Single page application made in Angular to archive films and rate them. Firestore used as backend; utilizing serverless model to reduce the barrier with implementing and maintenance of the app. Users can sign up, log in and rate films; only Admin users can add, edit and remove film entries (Complete CRUD). Bootstrap used as the front-end framework.
## PTBR
Projeto pessoal desenvolvido com o intuito de aplicar conceitos estudados. Trata-se de uma SPA para controle pessoal de filmes assistidos construído utilizando as tecnologias Angular, Firebase e Bootstrap. Ele contém um simples painel administrativo onde usuários podem se cadastrar, consultar e avaliar filmes. Somente usuários administradores podem utilizar o CRUD para adicionar, editar e remover filmes. Todo o backend foi implementado com Firestore, fazendo uso do severless e reduzindo o esforço necessário para a implementação e manutenção do app.
